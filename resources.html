AuraLit - Automated Academic Download Tool

Overview

AuraLit is a powerful automated academic literature download tool, meticulously designed for researchers, scholars, and students. Its core function is to batch process and download academic papers from a list of DOI links, offering support for a wide array of major academic publishers.

Key Features

  • Batch Downloading: Import DOI links from a CSV file for automated batch downloading.
  • Smart Publisher Recognition: Automatically identifies different publisher websites and applies appropriate download strategies.
  • Proxy Support: Built-in support for remote access proxies, enabling access to restricted resources via institutional VPNs.
  • Download Management: Supports pausing/resuming downloads and allows users to skip the current article at any time.
  • File Organization: Automatically organizes downloaded files, using the DOI as a standard filename.
  • Status Tracking: Tracks downloaded and undownloaded articles, supporting selective downloads.
  • User Interface: Features an intuitive graphical user interface (GUI) that displays download progress and logs in real-time.

Supported Publishers

AuraLit supports a broad range of publishers, including Elsevier (ScienceDirect), Wiley, American Chemical Society (ACS), Nature, Science, Royal Society of Chemistry (RSC), MDPI, Frontiers, PNAS, and other general academic websites.

System Requirements

  • Operating System: Windows, macOS, or Linux.
  • Browser: Google Chrome.
  • WebDriver: Chrome WebDriver (must match your Chrome version).
  • Access: Institutional credentials for accessing restricted resources.

Detailed installation and configuration guides (for `config.json` and DOI CSV format) are provided with the software.
